How to add Sunshine Console to your project:

Option #1 - Use NuGet

If you're using Visual Studio, Xamarin Studio, or MonoDevelop, you can easily grab Sunshine Console using NuGet.

  • For VS, go to Solution Explorer and right-click References, then select Manage NuGet Packages. Search for Sunshine Console and click Install.
  • For XS/MD, go to the Solution view and right-click Packages, then select Add Packages. Search for Sunshine Console, check its checkbox, and click Add Package.

NuGet will automatically add Sunshine Console (and OpenTK, which it requires) to your project.

Option #2 - Download the .zip

Go to the latest release and download SunshineConsole.zip. Extract it somewhere that your project can access it (I prefer to put it in the main project directory with my .cs files). Now it's time to add these files as references in your project.

  • For VS, go to Solution Explorer and right-click References, then select Add Reference. Go to the Browse tab and locate the files you just extracted. Add both SunshineConsole.dll and OpenTK.dll as references.
  • For XS/MD, go to the Solution view and right-click References, then select Edit References. Go to the .Net Assembly tab and click the Browse button near the bottom. Locate the files you just extracted and add both SunshineConsole.dll and OpenTK.dll.
